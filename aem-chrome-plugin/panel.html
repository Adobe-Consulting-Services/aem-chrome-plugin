    The Logger utility enables logging for a form and helps you debug form-related issues. One can configure Logger based on the requirements and set the level and category for it for any specific request. The log configuration is made up of the log level and the logger category.

    In the log URL, the log configuration query string parameter is defined as follows:{a level}-{b level}-{c level}-{d level}

    For example, to enable a logger to have the following settings : xfa level : INFO, AF Level : ALL and OFF for other categories, the value of window.AF_log_config should be "a4-b0-c0-d7" Please refer the tables below for the configuration :

    Log Categories

    Log Category Description
    a xfa (Scripting engine-related logs)
    b xfaView (Layout engine-related logs)
    c xfaPerf (Performance-related logs)
    d Adaptive Form
